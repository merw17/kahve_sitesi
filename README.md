Kahve Sitesi Projesi:
Bu proje, temel web teknolojileri kullanılarak geliştirilmiş modern ve kullanıcı dostu bir kahve dükkanı web sitesidir.


Özellikler:
Dinamik İçerik: JavaScript ile yönetilen etkileşimli öğeler.
Modern Tasarım: HTML5 ve CSS3 standartlarına uygun arayüz.
Responsive: Mobil cihazlarla uyumlu yapı. (Eğer yaptıysan bu maddeyi tutabilirsin).

Kullanılan Teknolojiler
HTML5: Sayfa yapısı ve içerik.

CSS3: Görsel tasarım ve layout.

JavaScript: Dinamik özellikler ve kullanıcı etkileşimi.

NPM: Paket yönetimi (Dosyalarındaki package.json içeriğine dayanarak).
